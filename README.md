# BScHons Data Science Academic Tracker

A high-performance academic management system designed to track progress, visualize performance, and forecast future grade requirements for Data Science undergraduates.

## 🚀 Features
* **Longitudinal Tracking:** Combo-chart visualization displaying Semester SGPA (bar) vs. Cumulative CGPA (line).
* **Target-Driven Engine:** Back-calculates the required average grade needed across remaining credits to hit specific Graduation Classes (e.g., 1st Class, 2nd Upper).
* **Curricular Ledger:** A structured grid matrix aligned with standard Data Science degree programs, allowing for easy updates of grades and credit tracking.
* **Responsive Dashboard:** Built with Tailwind CSS and Chart.js for a professional, dark-themed UI that works on desktop and mobile.

## 🛠 Tech Stack
* **Frontend:** HTML5, Tailwind CSS
* **Logic:** Vanilla JavaScript (ES6)
* **Visualization:** Chart.js
* **Data Persistence:** Browser LocalStorage (Auto-saves your progress locally)

## 📋 How to Use
1. Clone this repository or download the source files.
2. Open `index.html` in any modern web browser.
3. Use the "Interactive Curricular Matrix" to add or update your completed modules.
4. Set your graduation target in the "Prediction Engine" to see the required future performance.
5. Use the "Backup JSON" button to export your progress for safe keeping.
